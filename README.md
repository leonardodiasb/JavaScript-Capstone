## Built With

- HTML/CSS
- Javascript
- Webpacker
- NPM

## Getting Started

### Install

- `git clone https://github.com/leonardodiasb/JavaScript-Capstone.git`
- `cd JavaScript-Capstone`
- `npm install`
- `npm run start`
